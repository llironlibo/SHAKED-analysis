# Data directory

The patient-level data underlying this study are **not included** in this
repository. They contain protected health information and are available only
under controlled access. See the Data Availability statement in the published
article for the request and approval procedure.

To reproduce the analyses, place the de-identified analytic dataset here as:

```
data/cleaned/unified_pilot_cohort.csv
```

obtained through the controlled-access procedure described in the article.
Everything in this directory except this README is git-ignored, so data files
cannot be committed accidentally.
